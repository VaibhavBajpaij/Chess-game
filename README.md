# ♟️ Chess Game

A real-time multiplayer chess game built with Node.js, Express, and Socket.io.

## 🌐 Live Demo
Coming soon...

## ✨ Features
- Real-time multiplayer gameplay using WebSockets
- Full chess rules implementation
- Clean and responsive UI
- Two players can play from different browsers

## 🛠️ Tech Stack
- **Backend:** Node.js, Express.js
- **Real-time:** Socket.io
- **Templating:** EJS
- **Frontend:** JavaScript, CSS3

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm

### Installation
```bash
git clone https://github.com/VaibhavBajpaij/Chess-game.git
cd Chess-game
npm install
npm start
```

Open your browser at `http://localhost:3000`

## 🎮 How to Play
1. Open the app in two different browser tabs
2. First player gets **White** pieces
3. Second player gets **Black** pieces
4. Take turns making moves in real-time!
